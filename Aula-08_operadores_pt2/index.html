Operadores dessa aula

  • Relacionais
  • Lógicos
  • Ternario

Operadores Relacionais

  • (>) - Maior
  • (<) - Menor
  • (>=) - Maior Igual
  • (<=) - Menor Igual
  • (==) - Igual
  • (!=) - Diferente

    • Qualquer teste feito com Qualquer um dos operadores irá resultar em um valor Logico (Booleano - Verdadeiro ou Falso)

Exemplo

  • 5 > 2 -> True
  • 7 < 4 -> False
  • 8 >= 8 -> True
  • 9 <= 7 -> False
  • 5 == 5 -> True
  • 4 != 4 -> False

Operador de identidade

Operador de identidade -> (===)

  • 5 == 5 -> True
  • 5 == '5' -> True
  • 5 === '5' -> False
  • 5 === 5 -> True

Operadores Logicos

  • ! - Negacao
  • && - Conjuncao (e)
  • ||- Dijuncao (ou)
