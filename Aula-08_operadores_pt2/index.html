Operadores dessa aula

  • Relacionais
  • Lógicos
  • Ternario

Operadores Relacionais

  • (>) - Maior
  • (<) - Menor
  • (>=) - Maior Igual
  • (<=) - Menor Igual
  • (==) - Igual
  • (!=) - Diferente

    • Qualquer teste feito com Qualquer um dos operadores irá resultar em um valor Logico (Booleano - Verdadeiro ou Falso)

Exemplo

  • 5 > 2 -> True
  • 7 < 4 -> False
  • 8 >= 8 -> True
  • 9 <= 7 -> False
  • 5 == 5 -> True
  • 4 != 4 -> False

Operador de identidade

Operador de identidade -> (===)

  • 5 == 5 -> True
  • 5 == '5' -> True
  • 5 === '5' -> False
  • 5 === 5 -> True

Operadores Logicos

  • ! - Negacao
    • ! = True -> False (Uma coisa não verdadeira é falsa)
    • ! = False -> True (Uma coisa não falsa é verdadeira)
  • && - Conjuncao (e)
    • True && True -> True
    • True && False ->False
    • False && True -> False
    • False && False -> False
  • ||- Dijuncao (ou)
    • True || True -> True
    • True || False-> True
    • False || True -> True
    • False || False -> False

Ordem de Recedencia dos Operadores

  1. Aritmeticos - (),**, /
  2. Relacionais - >, <, >=
  3. Logicos - !, &&, ||
    1. ! - Negacao
    2. && - Conjunção
    3. || - Dijunção

Operador Ternario

Teste (?) True (:) False

    teste - Um Teste Logico

    (?) True - Oque vai acontecer se o teste for Verdadeiro

    (:) False - Oque Vai acontecer se o teste for Falso

media>=7.0 ? "Aprovado" : "Reprovado"
