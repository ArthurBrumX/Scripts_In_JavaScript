Operadores Aritméticos

Exemplo Operandos (5) e (2) - operadores Binarios

  • 5 (+) 2 = 7 -> Adicao
  • 5 (-) 2 = 3 -> Subtracao
  • 5 (*) 2 = 10 -> Multiplicacao
  • 5 (/) 2 = 2.5 -> Divisao Real
  • 5 (%) 2 = 1 -> Resto Da Divisao Inteira
  • 5 (**) 2 = 25 -> Pontecia

Ordem de precedencia dos operadores Aritmeticos - Ordem de Prioridades

  • "()" -> Parenteses
  • "**" -> Potencia
  • "*"", "/" , "%" -> Multiplicacao, Divisao, Resto da Divisao
  • "+", "-" ->Adição, Subtracao

Atribuição Simples

  • var a = 5 + 3 -> 8
  • var b = a % 5 -> 3
  • var c = 5 * b ** 2 -> 45
  • var d = 10 - a / 2 -> 6
  • var e = 6 * 2 / d -> 2
  • var f = b % e + 3 / e -> 3

Auto-Atribuições

Exemplo -> Var n = 3

  • n = n + 4 -> 7
  • n = n - 5 -> 2
  • n = n * 4 -> 8
  • n = n / 2 -> 4
  • n = n ** 2 -> 16
  • n = % 5 -> 1

----------------------------

Simplificando

  • n = n + 4 -> 7
    • Simplificado -> n+=4 Recebe ele mesmo Somado com o numero da frente
  • n = n - 5 -> 2
    • Simplificado -> n-=4 Recebe ele mesmo Subtraido com o numero da frente
  • n = n * 4 -> 8
    • Simplificado -> n*=4 Recebe ele mesmo Multiplicado com o numero da frente
  • n = n / 2 -> 4
    • Simplificado -> n/=4 Recebe ele mesmo Dividido com o numero da frente
  • n = n ** 2 -> 16
    • Simplificado -> n**=4 Recebe ele mesmo Elevado a pontência com o numero da frente
  • n = n% 5 -> 1
    • Simplificado -> n%=4 Recebe ele mesmo Resto da Divisão com o numero da frente

Incremento

Var x = 5

  • x = x + 1 -> 6 (A variavel recebe a variavel mais 1) -> simplificando ainda mais -> Operador de Incremento -> x++
    • x = x + 1 -> (x++)
  • x = x - 1 -> 5
    • x = x - 1 -> (x--)
  • x = x * 1 -> 5
    • x = x * 1 -> (x**)
  • x = x / 1 -> 5
    • x = x / 1 -> (x//)
  • x = x ** 1 -> 25
    • x = x * 1 -> (x****)
  • x = x % 1 -> 5
    • x = x % 1 -> (x%%)

Pré-Incremento = (n++)

Pós-Incremento = (++n)
